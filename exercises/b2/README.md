# B2: Git & GitHub Essentials — Exercise Files

These files accompany [Module B2: Git & GitHub Essentials](../../modules/b2-git-github.qmd).

## Contents

| File | Description |
|---|---|
| `git_exercise_walkthrough.md` | Step-by-step exercise where students create their own Git repo, make commits, inspect history, and optionally push to GitHub. Includes exact commands and expected output for each step. |
| `econ_gitignore` | A ready-to-use `.gitignore` template for economics research projects. Copy to your project root and rename to `.gitignore`. |
| `git_cheatsheet.md` | One-page reference for the most common Git commands, key concepts, and troubleshooting. |
| `good_commit_messages.md` | Examples of good vs. bad commit messages with a template and guidelines for economics projects. |

## How to Use

1. Start with `git_exercise_walkthrough.md` — work through it step by step in your terminal
2. Keep `git_cheatsheet.md` open as a reference while you work
3. Read `good_commit_messages.md` before writing your first commit messages
4. Copy `econ_gitignore` into your project and rename it to `.gitignore`

## Setup Requirements

- Git installed (Mac: `xcode-select --install`; Windows: [Git for Windows](https://gitforwindows.org/))
- Terminal access (see B1 exercises if you need setup help)
- A text editor (VS Code, Sublime Text, or any editor you are comfortable with)
- (Optional) A GitHub account for Part 5 of the walkthrough
