%% ====== SLIDE 23: EMILY — CREATING A VOICE FILE ======
\begin{frame}{Creating a Voice File}
\begin{columns}[T]
\begin{column}{0.47\textwidth}
\begin{tcolorbox}[accentbox]
\small
A ``voice file'' is a plain Markdown file that describes how you write --- tone, structure, habits. The AI reads it and matches your style.
\vskip4pt
\textbf{You don't write it from scratch.} Give the AI samples of your real writing --- emails, letters of recommendation, feedback --- and ask it to describe your patterns.
\vskip4pt
The result is a file like \texttt{EB-VOICE.md} that you can edit, refine, and reuse across projects.
\end{tcolorbox}
\end{column}
\begin{column}{0.47\textwidth}
\begin{tcolorbox}[colback=DarkSlate, colframe=DarkSlate, arc=3pt, left=6pt, right=6pt, top=4pt, bottom=4pt]
{\color{CoolGray}\ttfamily\tiny The prompt:\\[2pt]
\color{white}\scriptsize
Read these five emails I've sent.\\
Describe my writing style: tone,\\
sentence length, how I open and\\
close messages, what I avoid.\\
Save it as EB-VOICE.md.
}
