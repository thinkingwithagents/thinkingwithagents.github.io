What's here, and what comes next.
  1. Everything on these pages comes from one file — cv-data.json — generated from your CV.
  2. To update the site, edit cv-data.json (or drop in a new CV) and run python build.py.
  3. Add a headshot by setting site.photo; switch the live design by changing site.theme.
  4. Pick the direction you like best — that becomes your index.html, ready to deploy.
