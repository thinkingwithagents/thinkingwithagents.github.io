Getting Set Up

Tools for the AI Bootcamp

Option 1: Apps (easiest way to start)

These are point-and-click applications. Download, sign in, and start using them.

Claude (by Anthropic)

A conversational AI assistant. Available as a desktop app and in your browser.

  • Download: claude.ai/download (Mac and Windows)
  • Or use in browser: claude.ai
  • Subscription: Free tier available. Pro ($20/month) gives more usage and access to the most powerful models.
  • Create an account: claude.ai — sign up with your email

ChatGPT (by OpenAI)

Another widely-used AI assistant with desktop and mobile apps.

  • Download: openai.com/chatgpt/download (Mac and Windows)
  • Or use in browser: chatgpt.com
  • Subscription: Free tier available. Plus ($20/month) gives access to GPT-4o and more.

Microsoft Copilot (free for all UVM)

UVM provides enterprise-protected access to Microsoft Copilot at no cost.

  • Access: go.uvm.edu/copilot
  • Subscription: Free — included with your UVM account
  • Note: This is the safest option for working with student data or sensitive information, since it’s covered by UVM’s enterprise agreement.

Option 2: Command line tools (more powerful)

These run in your computer’s terminal — a text-based interface where you type commands. The terminal gives AI tools direct access to your files, which makes them much more powerful for real work.

What’s a terminal?

It’s an app that’s already on your computer:

  • Mac: Open Terminal (search for “Terminal” in Spotlight, or find it in Applications → Utilities)
  • Windows: Open PowerShell (search for “PowerShell” in the Start menu)

You type commands and press Enter. That’s it.

Installing Claude Code

Claude Code is an AI assistant that runs in your terminal. It can read your files, write documents, run code, and more.

Step 1: Install Node.js (a tool that Claude Code needs to run)

  • Go to nodejs.org
  • Click the big green LTS download button
  • Run the installer — accept all the defaults

Step 2: Install Claude Code

Open your terminal and type this command, then press Enter:

npm install -g @anthropic-ai/claude-code

Wait for it to finish (you’ll see some text scroll by — that’s normal).

Step 3: Launch it

Type this and press Enter:

claude

It will ask you to sign in to your Anthropic account. Follow the prompts.

Subscription: Claude Code requires a Claude Max ($100/month) or Pro ($20/month + usage-based API billing) subscription.

Installing OpenAI Codex

Codex is OpenAI’s terminal-based AI tool, similar to Claude Code.

Step 1: Install Node.js (same as above — skip if you already did this)

  • Go to nodejs.org → download LTS → run installer

Step 2: Install Codex

npm install -g @openai/codex

Step 3: Launch it

codex

Subscription: Requires an OpenAI API account with billing enabled.

Resources

Some extra tools that aren’t required but are nice to have:

  • Typora ($14.99) — A clean markdown editor. AI tools often output markdown files; Typora makes them look polished for reading and sharing.
  • Ghostty (free) — A fast, modern terminal app. Looks nicer than the default Terminal/PowerShell. Purely cosmetic.
  • VS Code (free) — A code editor. Claude Code and GitHub Copilot both have VS Code extensions if you prefer a graphical interface over the terminal.
