Getting Set Up

Tools for the AI Bootcamp

Two categories of AI tools

For this bootcamp, the most useful distinction is not “browser vs. app.” It is:

  • Chat-based tools: good for asking questions, drafting text, analyzing uploaded files, and trying quick prompts.
  • Agentic tools: good for working on files, code, and multi-step tasks with direct access to your project or local environment.

Chat tools can still write code snippets or comment on code. What they usually do not do is directly search your folders, edit local files, or run commands on your machine. That is the role of the agentic tools we are focusing on for the terminal demos.

Option 1: Chat-based tools (easiest way to start)

These are point-and-click applications. Download, sign in, and start using them.

Claude (by Anthropic)

A conversational AI assistant. Available as a desktop app and in your browser.

Best for:

  • drafting and revising text
  • brainstorming
  • summarizing uploaded files
  • generating code snippets or examples

Not the same as Claude Code:

  • Claude in the browser or desktop app can help you talk through code and files

  • Claude Code can directly work in your terminal, read local project files, edit files, and run commands

  • Download: claude.ai/download (Mac and Windows)

  • Or use in browser: claude.ai

  • Subscription: Free tier available. Pro ($20/month) is the most straightforward paid option. Max starts at $100/month for substantially more usage.

  • Create an account: claude.ai — sign up with your email

ChatGPT (by OpenAI)

Another widely-used AI assistant with desktop and mobile apps.

Best for:

  • drafting and revising text
  • summarizing uploaded files
  • brainstorming
  • generating code snippets or examples

Not the same as Codex:

  • ChatGPT is primarily a chat interface

  • Codex is OpenAI’s coding agent, designed to work in your terminal, code editor, or Codex app with direct project actions

  • Download: openai.com/chatgpt/download (Mac and Windows)

  • Or use in browser: chatgpt.com

  • Subscription: Free tier available. Plus ($20/month) is the standard paid tier for most users.

UVM note: Microsoft Copilot

UVM provides enterprise-protected access to Microsoft Copilot Chat through your work account.

  • Access: go.uvm.edu/copilot
  • What it’s good for: low-friction web chat and some Microsoft 365 workflows
  • What it is not for: the terminal/file-access demos we are centering in this bootcamp
  • If you want deeper Office integration: ask about a Microsoft 365 Copilot license, which adds stronger Word, Excel, PowerPoint, and Outlook integration
  • Why it’s not the main setup path here: for this bootcamp, we want tools that can clearly demonstrate the difference between chat-based help and agentic file/code workflows

Option 2: Agentic tools (more powerful)

These run in your computer’s terminal — a text-based interface where you type commands. The terminal gives AI tools direct access to your files, which makes them much more powerful for real work.

What’s a terminal?

It’s an app that’s already on your computer:

  • Mac: Open Terminal (search for “Terminal” in Spotlight, or find it in Applications → Utilities)
  • Windows: Open PowerShell (search for “PowerShell” in the Start menu)

You type commands and press Enter. That’s it.

Installing Claude Code

Claude Code is an AI assistant that runs in your terminal. It can read your files, write documents, run code, and more.

This is the Anthropic tool that most clearly shows the difference between:

  • a chat tool that answers questions
  • an agentic tool that can act on a project

Step 1: Install Node.js (a tool that Claude Code needs to run)

  • Go to nodejs.org
  • Click the big green LTS download button
  • Run the installer — accept all the defaults

Step 2: Install Claude Code

Open your terminal and type this command, then press Enter:

npm install -g @anthropic-ai/claude-code

Wait for it to finish (you’ll see some text scroll by — that’s normal).

Step 3: Launch it

Type this and press Enter:

claude

It will ask you to sign in. Follow the prompts.

How you pay for it:

  • Claude Pro ($20/month) or Claude Max (from $100/month) lets you sign in with your Claude account
  • Anthropic also supports logging in with an Anthropic Console account if you prefer API-based billing instead of a Claude subscription

For most faculty trying this on their own, Pro is the default place to start.

Installing OpenAI Codex

Codex is OpenAI’s coding agent, similar in spirit to Claude Code.

Step 1: Install Node.js (same as above — skip if you already did this)

  • Go to nodejs.org → download LTS → run installer

Step 2: Install Codex

npm install -g @openai/codex

Step 3: Launch it

codex

How you pay for it:

  • The simplest path is to sign in with an active ChatGPT subscription
  • You do not need a separate API account just to try the subscription-based Codex workflow

For most faculty, ChatGPT Plus is the simplest paid entry point if they want to try Codex.

Resources

Some extra tools that aren’t required but are nice to have:

  • Ghostty (free) — A fast, modern terminal app. Looks nicer than the default Terminal/PowerShell. Purely cosmetic.
  • VS Code (free) — A code editor. Claude Code and GitHub Copilot both have VS Code extensions if you prefer a graphical interface over the terminal.

Markdown viewers / editors

AI tools often output rough-looking .md files. A good Markdown viewer or editor makes them render quickly as readable, attractive text.

  • Typora ($14.99) — Recommended default. It gives a live, clean preview and works well as a standalone app for opening Markdown files directly.
  • MarkText (free) — A simple open-source Markdown editor for macOS, Windows, and Linux.
  • Zettlr (free) — A stronger option for larger academic writing projects if you want more structure and writing features.
  • MacDown (free, macOS only) — A lightweight Mac-native Markdown editor worth considering.

If you just want one app that opens Markdown files and makes them look good immediately, Typora is the default recommendation here.
