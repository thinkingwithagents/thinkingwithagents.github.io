Outside Resources

Talks, guides, and links beyond the bootcamp itself

This page is a running list of useful outside resources: talks, articles, demos, explainers, and people worth following. It is broader than the bootcamp itself.

How to use this page

For each link, try to add:

  • what it is
  • why it is useful
  • who it is for
  • whether it is more about teaching, research, or general AI literacy

Talks and explainers

Paul Goldsmith-Pinkham

  • Getting started with Claude Code A good starting point for faculty who want to understand what Claude Code is for, how it fits into a workflow, and why working in the terminal feels different from ordinary chat.

Claude Blattman

  • claudeblattman.com Useful for skills, templates, and Claude Code-oriented workflow examples. Good for people who want concrete patterns rather than abstract prompting advice.

Obsidian

  • Obsidian A flexible notes app that works well as a personal knowledge base, project notebook, or place to store prompts, reusable instructions, and session notes.
  • Obsidian Help Useful if you want to learn linking, vault structure, templates, and other features for organizing AI-assisted work over time.
